## 🚀 Getting Started

### 1️⃣ Clone the Repo

```bash
git clone https://github.com/SyedDanish6897/portfolio.git
cd portfolio
```

### 2️⃣ Install Dependencies

Make sure you have **Python 3.x** and Flask installed:

```bash
pip install flask
```

### 3️⃣ Run the App

```bash
python app.py
```

Open in browser → `http://127.0.0.1:5000/`
